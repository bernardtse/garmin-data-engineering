Garmin Device User Fitness Data API

Welcome to our API. This API provides exercise data generated by an individual user over a number of years from their Garmin watch. You can access all our endpoints for visualisation purposes.

The data field names are the same as the ones used by Garmin. They are defined here.

The documentation and code for this page is available on our GitHub repository (original) / (fork).

The basic endpoints return all the available data for that table. The additional endpoints return filtered data.

Type of Activities

This endpoint provides data on the types of activities undertaken.
Each activity has a unique identifier.
The endpoint provides all the identifiers.

Basic Endpoints
  • /api/activity-types
Activities

These endpoints provide data on the activities undertaken, including type, duration, etc.
Basic endpoints provide all data, while additional endpoints allow filtering by activity type, last N records, and dates.

Basic Endpoints
  • /api/activities/all
